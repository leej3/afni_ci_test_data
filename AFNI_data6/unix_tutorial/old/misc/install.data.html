Installing the AFNI_data6 class data tree under the $HOME directory

   In a typical AFNI bootcamp, the AFNI_data6 directory will be put under the
   $HOME directory.

   The following commands will download it and place it in the $HOME directory:

        cd
        curl -O http://afni.nimh.nih.gov/pub/dist/edu/data/AFNI_data6.tgz
        tar xzf AFNI_data6.tgz


   Data installation is part of setting up a computer for an AFNI bootcamp.
   For full details, see:

        howto: class setup
