Installing AFNI

   To install AFNI, please see HowTo #0.
   For help with installation difficulties, please search or post to the AFNI Message Board.

   To update AFNI (or possibly do an initial install of binaries), consider:

        @update.afni.binaries -defaults
        help: @update.afni.binaries


   For a (moderately) current linux install, see: HowTo #0: install AFNI on Linux.
   For a current mac install, see: HowTo #0: install AFNI on OS X 10.7 or 10.8.

   To access any of the "HowTo#0 : install" pages, see: howto #0: install AFNI

   To set up a computer for an AFNI bootcamp, see:

        howto: class setup
